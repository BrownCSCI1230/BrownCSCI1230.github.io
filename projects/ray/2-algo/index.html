Shadow generation

To add shadows to your renders, you need to do recursive raytracing. When the ray(view ray) hits a surface, you need to shoot another ray(shadow ray) from the intersection point to a light source to check intersections in the scene again. If there are intersections, that means some objects are blocking the light source.

Falloff function

Falloff is the effect of lighting intesity gradually fading as the illuminated point gets farther from the light source. Think about the case of a spotlight, which has a spread range with the shape of a cone. Only objects within the spotlight cone receive light. Within this cone there is a falloff effect where the light intensity gets weaker when it diverges from the direction of the spotlight.
