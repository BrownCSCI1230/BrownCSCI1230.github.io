Introduction

Welcome to CS 1230!

The purpose of this lab is twofold: to set you up with what you need to work on assignments locally, and to help you learn the basics of C++, the programming language that we will be using in this course.

In the rest of this lab, you will:

  1. Install Qt 6 & Qt Creator locally on your computer,
  2. Build and run a starter Qt program, and
  3. Learn about the basics (and the subtleties) of C++.

Before You Begin

We assume you're already familiar with using Github Classroom to accept assignments, Ed to ask questions, and Gradescope to submit work. Before proceeding to the next section, make sure you have done the following:

  • Joined our course's Ed page,
  • Enrolled in CS 1230 on Gradescope,
  • Accepted this lab's assignment from Github Classroom, and
  • Cloned the resulting repository to your local machine.

All set? Let's get started!

Qt and Qt Creator

In CS123, we use Qt and Qt Creator to develop, build, and run all our projects and labs. Here's a brief description of each:

  • Qt is a software used for building graphical user interfaces (GUIs) and cross-platform applications, e.g. for smart TVs or in-vehicle displays.
  • Qt Creator is the integrated development environment (IDE) included with each Qt install. It provides useful tools for developing in C++, which you'll learn about later in the course.

In this section, we'll walk you through both installing Qt and Qt Creator to your local machine and opening your first project. We'll also give you a quick tour of the Qt Creator interface.

But I prefer developing in VS Code / Emacs / Notepad++ / etc!

If you know what you're doing, you may certainly write code in your IDE of choice. In the past, students have successfully used other IDEs for writing code, before using Qt Creator to build and run their projects.

That being said, the CS 1230 course staff will be using Qt Creator to grade your assignments. Thus, you'll probably want to install Qt Creator anyway, if only to test that your code works as expected when running with it.

What is a .pro file?
A .pro file is a multi-platform project file, which Qt can turn into a platform-specific makefile. It stores configuration information, e.g. what files to include.
